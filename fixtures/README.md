# Fixtures

Small committed samples that let the whole pipeline run in tests and replay
without network access.

| File | Origin | Notes |
|---|---|---|
| `warn_sample.csv` | real — first rows of the NY DOL WARN Tableau CSV export (public record) | captured 2026-06-11 |
| `socrata_snap_sample.json` | real — NYC Open Data 5awp-wfkt SoQL aggregate (public domain) | captured 2026-06-11; note the genuine `Staten Island`/`Staten_Island` vocabulary drift |
| `acs_sample.json` | **synthetic**, format-accurate (the Census API now requires a key even for samples) | matches the ACS array-of-arrays shape exactly |
| `mmg_sample.csv` | **synthetic-derived** plausible county rates | MMG is registration-gated and not redistributable wholesale — do NOT commit real MMG exports |
| `pulse_sample.csv` | **synthetic**, in the normalized Pulse CSV shape the adapter expects | wave URLs churn; see `household_pulse.rs` |
| `golden/` | replay goldens generated by `cargo run -p replay -- --update-golden` | regenerate deliberately, in PRs |
